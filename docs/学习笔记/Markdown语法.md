# 1. 标题语法
> 标题是文档的骨架，分为多级，四级标题的字号大小与普通文本接近
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
普通文本
##### 五级标题
###### 六级标题
# 2. 段落语法
> 段落之间要空一行

第一段

第二段

第三段
# 3. 换行语法
> 换行只需要在文本末尾加两个空格，然后按回车键

第一行  
第二行
# 4. 强调语法
> 分为粗体和斜体，可以用`*`也可以用`_`，为了方便统一使用`*`。  
> 斜体左右一颗星，粗体左右两颗星，粗斜体三颗星。

*斜体*

**粗体**

***粗斜体***

# 5. 引用语法
> 引用前加一个`>`即可

> 这就是引用
# 6. 列表语法
> 分为有序和无序，有序用数字，无序用星

## 有序列表
1. 第一
2. 第二
3. 第三
## 无序列表
* 苹果
* 香蕉
* 橘子

`单词`

```java

    System.out.println("Hello World");
```
